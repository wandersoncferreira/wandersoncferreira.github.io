Improving Vim speed

May 15, 2020
clojure vim

Youtube

Speaker

Thoughbot

Thoughts

Vim is great tool. I learned a lot from the mindset of vim users about efficiency in editing text. I think there is not other place where you can find advices and people obsessed with fewer keystrokes.

Invest time to sharp your editing skills!

Recommendations

  1. Stay away from insert mode. In Emacs I should use more commands to movement and editing.
  2. Stop using arrow keys. In Emacs I already do not do that too much now.
  3. Learn different move characteres
  4. If you hold any character consider this a smell
  5. Use more F and T to movement. In Emacs it should be through jump-char package.
  6. Use the repeat command. I need to check this one out in Emacs, I never used it.
  7. Relative movements between lines should like a good idea in Emacs? I don’t think so right now.
  8. Use some kind of fuzzy finder. Right now I am using Projectile and Ido mode that cover almost all my use cases right now.
  9. Look how to use Vim Surround in Emacs, how to do that?The answer might be wrap-region package.
